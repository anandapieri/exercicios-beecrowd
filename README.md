
# Exercícios Beecrowd

Repositório com soluções de exercícios da plataforma Beecrowd desenvolvidas em Python para prática de lógica de programação, algoritmos e estruturas de dados.

## Objetivo

Este repositório foi criado com o objetivo de:

- praticar lógica de programação;
- desenvolver raciocínio algorítmico;
- estudar estruturas de dados;
- aprimorar manipulação de strings, vetores e matrizes;
- acompanhar minha evolução em programação com Python.

## Tecnologias utilizadas

- Python 3
- Git
- GitHub

## Como executar os exercícios

1. Clone o repositório:

```bash
git clone LINK_DO_REPOSITORIO
```

2. Acesse a pasta:

```bash
cd exercicios-beecrowd
```

3. Execute um exercício:

```bash
python strings/1238-combinador.py
```
